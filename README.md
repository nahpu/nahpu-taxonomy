# nahpu-taxonomy

This is a work in progress to provide a baseline taxonomy for the NAHPU digital catalog. It simplifies importing taxonomic data into the app. This project serves two purposes:

1. Allow experts to update the taxa based on a widely used taxonomic database for a given taxon group. For example, [the Clements checklist](https://www.birds.cornell.edu/clementschecklist/) for birds and [the ASM Mammal Diversity Database](https://www.mammaldiversity.org/) for mammals.
2. Harmonize the database data format for easy reading within the app.

## Motivation

In an ideal world, we will query the taxonomy directly from the source and import it into the [NAHPU](https://nahpu.app] app. Unfortunately, the taxonomic database data format is inconsistent. Some are not version-controlled, which makes it difficult to track changes. This effort ensures that we provide a consistent taxonomic dataset for the app.
